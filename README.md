# Argo-rollout-demo

This DEMO demonstrates how to use Argo Workflow , Argo CD and Argo Rollout to create a complete Microservice CI/CD workflow with different Rollout Strategies (Canary/Blue-Green) in a kubernetes Cluster. 

Tech Stack:
- Kubernetes
- Quarkus
- Docker
- Tilt.dev
- Kaniko 
- Argo Workflow
- Argo CD
- Argo Rollout
